# Brand assets

This folder is where local integration icon/logo assets go, per the
HA 2026.3+ local-brands mechanism (`home-assistant/brands` no longer
accepts pull requests for custom integrations).

`icon.png`/`icon@2x.png` are currently a **placeholder** (a plain
"EP30" wordmark), added only to satisfy HACS's brand-asset validation
check -- replace with real artwork before any 1.0/public announcement.
